== Description ==

This plugin will show you information about your website visitors: country, city, region, ISP. It will also show a map with the number of visitors from every country. The plugin uses the API from https://www.ipgp.net

== Installation ==

This section describes how to install the plugin and get it working.

1. Download the plugin files and upload them into your wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. On the bottom of the admin menu select "IPGP". You will be directed to the plugin main page
4. On the page, follow the link to get your API key. Then insert the given API key into the field and click "Save"
5. Once your visitors will be recorded they will be shown on the page.
